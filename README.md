 ## 🚀 Tesla Social Media Analytics

A data analytics project focused on evaluating social media performance for Tesla using Microsoft Excel. The project applies data cleaning, exploratory analysis, and KPI tracking to derive actionable marketing insights.

### 📊 Key Insights
- Identified top-performing platforms based on engagement rates  
- Determined high-impact content types and posting strategies  
- Evaluated hashtag effectiveness to improve reach and interaction  
- Analyzed campaign performance and ROI  
- Tracked follower growth trends and retention patterns  

### 🛠 Tools & Techniques
- Microsoft Excel (Pivot Tables, Charts, Data Cleaning, Formulas)  
- KPI Analysis & Data Visualization  

### 🎯 Outcome
Delivered data-driven recommendations to optimize content strategy, improve engagement, and enhance campaign effectiveness.
